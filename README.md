# Arduino-BlackBox
Arduino based CAN-Bus Diagnostic Tool

** ID-Config for SMART CDI 2003 **

## Introduction

# I2C Pinout

* SDA: ANALOG4
* SCL: ANALOG5
* GND: GND
* VCC: 5V

# MCP2515 Pinout

* INT: PIN2
* SCK: PIN13
* MOSI: PIN11
* MISO: PIN12
* CS: PIN10
* GND: GND
* VCC: 5V
